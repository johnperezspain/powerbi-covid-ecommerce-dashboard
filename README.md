# Impacto COVID-19 y rendimiento comercial (ecommerce) — Power BI

Proyecto final (Tokio School). Integración de datos COVID (OWID) y ventas ecommerce (Kaggle), modelado con calendario y medidas DAX para KPIs.

## Tecnologías
- Power BI
- Power Query
- DAX

## Contenido
- `Informe_para_el_medio_de_comunicación.pdf` – informe ejecutivo
- `informe_metodológico.pdf` – documentación metodológica

## Fuentes de datos
- Our World in Data (COVID): cases_deaths.csv
- Kaggle (ecommerce): ventas por factura
> Nota: por tamaño/licencia, los datasets no se incluyen en el repositorio.
